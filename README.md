# GeminiAgent 🤖

Web application chatbot berbasis **Gemini 1.5 Pro** dengan kemampuan AI Agent via **Composio** — dibangun dengan Next.js 15, TypeScript, dan Tailwind CSS.

---

## Tech Stack

| Layer      | Teknologi |
|------------|-----------|
| Framework  | Next.js 15 (App Router) |
| Language   | TypeScript 5.7 |
| Styling    | Tailwind CSS 3.4 |
| AI Model   | Google Gemini 1.5 Pro via `@google/genai` v1 |
| Agent Tools| Composio via `composio-core` |
| Streaming  | Server-Sent Events (SSE) |

> Setup: `npm install` lalu buat `.env.local` dengan GEMINI_API_KEY dan COMPOSIO_API_KEY
